# ZenSpace | Mindful Self-Help & Habit Center

ZenSpace is a premium, modern mindfulness hub designed to help you build positive daily intentions, track streaks, write in a gratitude canvas, practice paced breathing, and listen to calming ambient soundscapes generated directly in your browser.

## 🚀 Live Demo
Visit the deployed application live at: **[https://syedabersabil.github.io/doop/](https://syedabersabil.github.io/doop/)**

## 🛠️ Key Features
- **💨 Guided Breathing Sphere**: A visually animated lung simulator helper (inhale, hold, exhale) designed to center your mind.
- **🌿 Daily Intention checklist**: Add daily habits, track streaking milestones, and see dynamic progress updates.
- **📝 Gratitude Journal Canvas**: Write entries dynamically to shift focus to positive aspects of your day.
- **✨ Motivational Affirmations**: Get positive quotes from mental health guides and philosophical writers.
- **🎵 Ambient Sound Synthesizer**: Native Web Audio API synth to generate Binaural Singing Bowl frequencies and Ocean Wave pink-noise dynamics offline.
- **🌓 Zen Theme Toggle**: Light (Calming Sage) and Dark (Deep Pine Forest) layouts.

## 💻 Tech Stack
- **Framework**: Vue 3 (Composition Setup API)
- **Build Tool**: Vite 8
- **Audio synthesis**: Native Web Audio API Nodes
- **Deployment**: Github Pages

## ⚙️ Development Guide

First install npm dependencies:
```bash
npm install
```

Run in development mode:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Deploy build output directly onto Github Pages:
```bash
npm run deploy
```
